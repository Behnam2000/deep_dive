Jupyter notebooks let you mix **Markdown cells** and **Code cells**.

Here‚Äôs how to create a title like that.

---

## Step-by-step (the usual way)

1. **Insert a new cell**
2. Change the cell type to **Markdown**

   * Press `Esc` ‚Üí `M` (keyboard shortcut), or
   * Use the toolbar dropdown: `Code` ‚Üí `Markdown`
3. Type:

```markdown
# Functions
```

4. Press **Shift + Enter**

You‚Äôll get a large title exactly like in your screenshot.

---

## Header sizes (Markdown levels)

```markdown
# Functions        ‚Üê biggest (section title)
## Functions       ‚Üê subsection
### Functions      ‚Üê smaller
#### Functions
```

Use:

* `#` for main sections
* `##` for subsections

---

## Pro tip: Fast keyboard flow

While coding, you can do this quickly:

1. Press `Esc` (command mode)
2. Press `B` (new cell below)
3. Press `M` (make it Markdown)
4. Type `# Functions`
5. Press `Shift + Enter`

Super fast once it‚Äôs muscle memory.

---

## Bonus: Styling (optional)

You can add emphasis if you want:

```markdown
# Functions
---
```

The `---` adds a horizontal line under the title.

---

## Mental model

* **Code cell** ‚Üí Python runs
* **Markdown cell** ‚Üí documentation, titles, notes, math, explanations

Good notebooks mix both.
